Mini Browser

Version 1.0 — Web 1.0 Edition

A minimal, text-focused web browser that strips away the modern web to reveal the raw content underneath.

No JavaScript. No CSS. No images. Just HTML.

Features: Back/Forward history, bookmarks, text-only mode, clickable links, form support, text file viewer, internal pages.

Built with: Plain HTML, CSS, and JavaScript — no frameworks, no libraries.

Proxy: Uses public CORS proxies to fetch and sanitize web pages.
