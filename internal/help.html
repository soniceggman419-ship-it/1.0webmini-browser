Help

What is this?

A minimal Web 1.0 style browser that strips away JavaScript, CSS, images, and media from web pages — leaving only the raw HTML content. It gives you a fast, text-focused browsing experience like the early web.

Entering a URL

Type a URL into the address bar and press Enter or click Go. If you don't type https://, it will be added automatically.

Examples:
https://en.wikipedia.org/wiki/World_Wide_Web
example.com

Internal Pages (int://)

You can access built-in pages using the int:// protocol:

  • int://welcome — Start page with URL input
  • int://help — This help page
  • int://about — About the browser

All internal pages are stored in the internal/ folder. Typing int://pagename loads internal/pagename.html.

Features

  • Back / Forward — Navigate your browsing history
  • Bookmarks — Click the ★ button to save the current URL. Saved bookmarks appear in the bar below the toolbar. Click a bookmark to go there.
  • Text Mode — Toggle "Text Mode" to strip all HTML and view only the plain text content of a page.
  • Status Bar — Shows loading progress and error messages at the bottom.
  • Clickable Links — Links on loaded pages are clickable and will navigate within the browser.

Text / Code Files

If the URL ends in a text or code file extension (.txt, .c, .java, .py, .js, etc.), the browser will display the raw file contents instead of trying to render it as HTML.

Limitations

  • No JavaScript — pages that rely on JS will not work
  • No CSS — pages will appear unstyled
  • No images, video, or audio
  • Some sites may block the proxy and fail to load
